## Sino-Korean System

- 공, 영 = 0
- 백 = $10^2$, ร้อย
- 천 = $10^3$, พัน
- 만 = $10^4$, หมื่น
- 억 = $10^8$, ร้อยล้าน
- 조 = $10^{12}$, ล้านล้าน, trillion

## Native Korean Number system

- 하나 = 1
- 둘 = 2
- 셋 = 3
- 넷 = 4
- 다섯 = 5
- 여섯 = 6
- 일곱 = 7
- 여덟 = 8
- 아홉 = 9
- 열 = 10
- 스물 = 20
- 서른 = 30
- 마흔 = 40
- 쉰 = 50
- 예순 = 60
- 일흔 = 70
- 여든 = 80
- 아흔 = 90

## Counting Unit

- 명 = person
- 개 = pieces (for things)
- 시 = hour (used for time)
- 마리 = animals
- 권 = Books
- 병 = Bottles
- 잔 = Cups/Glasses
- 인분 = Portion (of food)
- 벌 = Clothes
- 켤레 Pair (of clothes)
- 대 = Cars and Machines
- 장 = แผ่น (of paper)
- 조각 = Pieces of larger whole (e.g., cake, pie, pizza)
- 번 = Number/Times (e.g., 한번 = Once)
- 등 = Ranking
- 급 = Grade/Class
