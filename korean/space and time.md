# Relative Position

- 이것 = อันนี่
- 그것 = อันนั่น
- 저것 = อันโน่น
- 여기 = ที่นี่
- 저기 = ที่นั่น
- 거기 = ที่โน่น

## Prepositions

| Word | Meaning |
| ---- | ------- |
| A1   | B1      |
| A2   | B2      |
| A3   | B3      |

## Relative Time

- 지금 = now
- 요새, 요즘 = recently, nowadays
- 매일 = everyday
- 오늘 = today
- 어제 = yesterday
- 내일 = tomorrow
